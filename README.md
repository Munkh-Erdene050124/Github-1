Ашиглах заавар

Функцууд нь тооцоолол хийж үр дүнг буцаана.
run_tests() функц нь бүх функцуудыг ажиллуулж, дараах байдлаар үр дүнг шалгана:
CORRECT – Функц зөв ажилласан
WRONG – Функц ажилласан боловч буруу үр дүн гаргасан
NOT ATTEMPTED – Функцийг сурагч оруулаагүй, pass байна

Жишээ:

result = check_addition(2, 3)
print(result)  # 5

run_tests() – бүх функцуудыг шалгаж, оролдлого болон зөв эсэхийг мэдэгдэх
